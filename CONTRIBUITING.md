# Contributing

Gracias por tu interés en contribuir.

## Cómo contribuir

1. Haz fork del repositorio
2. Crea una rama: `feature/nombre`
3. Realiza tus cambios
4. Haz commit descriptivo
5. Envía un Pull Request

## Estándares

- Código en inglés
- Comentarios claros
- Mantener coherencia en estructura
- Documentar nuevas funciones

## Reportar errores

Abrir un Issue describiendo:
- Qué ocurre
- Qué debería ocurrir
- Capturas si es posible
